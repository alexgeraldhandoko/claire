# Claire

## Introduction

**Claire** is a financial model built for research/testing that is aimed 
at making accurate LOB predictions for the purpose of performing profitable
trading decisions. It is partly inspired by modern electronic market-makers
and will eventually be used to make live paper trading decisions that impact
the PnL of a mock trading account. It also learns and builds on previous LOB
research.

## Project Direction
1. Dataset collection
2. Dataset preprocessing
3. Model building
4. Model training, testing, and improvement
5. Trading algorithm
6. Live paper trading - model performance evaluated by PnL
More details can be read in Section 0 of [`Report.md`](./Report.md)
